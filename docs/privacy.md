# Privacy Policy

*Last updated: 2026-07-10*

## Summary

leveldb-forensic is a local Rust library and CLI. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

leveldb-forensic reads only the LevelDB files you point it at. All processing happens in memory on your local machine. Nothing is uploaded anywhere. It opens evidence read-only, never takes the database `LOCK`, and never writes to the directory.

## Telemetry

leveldb-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

leveldb-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/leveldb-forensic](https://github.com/SecurityRonin/leveldb-forensic).
